Salestown is a new app for Android and iOS that allows sales in your city to find you.

Fashion fans can even go for bargains and share their best finds with the Salestown community.

Salestown will be launched later this year in Winterthur. Other Swiss cities will follow soon.

  • IOS
  • Android
  • Node.js

The project goal was to develop basic prototype to check the idea.

Technologies we used:

iAndroid mobile applicaton: Retrofit - strong network request library, Sugar ORMiOS mobile applicaton: Objective-C

Server side:

Node.js, JavaScript, MySQL, REST, JSON, Angular.js, Bootstrap

Project stages:
  • Projection
  • Development
  • Testing
  • Submission & Deployment

Salestown project duration is from January 2016 to April 2016.

We were working using Fixed Price model based on classic project methodology.

Instead of creation prototype, we have developed full product with clean code.

After first release client plans to continue the development of project

by adding new functionality and attract users
