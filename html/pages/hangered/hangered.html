Hangered is social community to share, inspire, discover, save and shop all favourite boutiques and stores. User can browse and share products from anywhere and share favourite styles and designs.

The project goal was to create product MVP to rise investments.

Technologies we used

iOS mobile application: Objective C, UIKit, Networking
Server side: Node.js, JavaScript, MySQL, REST, JSON, Angular.js, Bootstrap

Project stages

user interface

Hangered project duration is from June 2015 to present. We were working using Time&Material model based on Scrum project methodology. During this time Customer gave positive feedback to our team that consists of Node.js developer, iOS developer, designer, front-end developer, Scrum Master and Account Manager.
